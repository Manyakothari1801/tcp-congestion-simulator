# Simulation & Predictive Analysis of TCP Congestion Control Algorithms

## Overview
This project explores how different TCP congestion control algorithms behave under varying network conditions. It combines simulation with machine learning to predict the most suitable algorithm.

## Algorithms Simulated
- TCP Reno  
- TCP CUBIC  
- TCP Vegas  
- TCP BBR  
- TCP LEDBAT  

## Key Features
- Network simulation engine for throughput, RTT, loss analysis
- Performance comparison across multiple algorithms
- ML-based prediction engine for selecting optimal algorithm
- Visualization of results using interactive charts

## Tech Stack
- Backend: Python, Flask
- Machine Learning: Scikit-learn, XGBoost
- Frontend: React, Tailwind CSS, Recharts
- Data Processing: NumPy, Pandas

## What I Learned
- Deep understanding of TCP/IP and congestion control
- Performance evaluation metrics (fairness, efficiency, convergence)
- Applying ML to systems-level problems
- Designing modular simulation architectures
